# SITCON 2024 Closing Video

## Getting Started

```
npm i
```

## Commands

Start the Next.js dev server:

```
npm run dev
```

Open the Remotion Studio:

```
npm run remotion
```

You should run this script after:

- changing the video template
- changing `config.mjs`
- upgrading Remotion to a newer version

Render the video:

```
npm run render
```
